Academic Reference

Robotics Kinematics Manual

Comprehensive mathematical formulas and specifications of the Lumina Precision simulator.

functions 1. Forward Kinematics (Denavit-Hartenberg Convention)

Forward kinematics (FK) computes the homogeneous matrix transformation $T$ mapping the robot coordinates from local tool center point frame (TCP) back to the base reference coordinates.

Each joint $i$ defines a local coordinate transformation matrix $A_i$ dependent on four structural DH Parameters:

DH Transformation Matrix Transformation Equation:

A_i = Rot_z(θ_i) * Trans_z(d_i) * Trans_x(a_i) * Rot_x(α_i)

Which expands into the standard matrix format:

[ cos(θ)   -sin(θ)·cos(α)   sin(θ)·sin(α)   a·cos(θ) ]
[ sin(θ)    cos(θ)·cos(α)  -cos(θ)·sin(α)   a·sin(θ) ]
[   0          sin(α)          cos(α)          d     ]
[   0            0               0             1     ]

calculate 2. Numerical Inverse Kinematics (CCD Solver)

Inverse Kinematics (IK) computes appropriate set of joint parameters necessary to locate the robot tip at desired coordinates $(X, Y, Z)$ in 3D cartesian coordinates.

Our real-time engine implements the highly convergent Cyclic Coordinate Descent (CCD) algorithm. This algorithm traverses backwards from wrist joints down to the base joint, minimizing the coordinate error vector iteratively:

CCD Angular Step Rotation Vector Update Formula:

φ = arccos( (V_c · V_t) / (|V_c| · |V_t|) )

Where:

  • $V_c$ is the vector pointing from active joint origin to active TCP location.
  • $V_t$ is the vector pointing from active joint origin to goal coordinate position target.
  • Joint angles are clamped inside safe limits: $J_i^{min} \le \theta_i \le J_i^{max}$.

scatter_plot 3. Workspace Monte Carlo Sampling

The total envelope workspace boundary represents all coordinate points in 3D space which are physically within the reach range of the robot gripper.

We compute this dynamically using a Monte Carlo probabilistic model generator. Given random joints rotations vectors generated evenly:

θ_i = J_i^min + r_i · (J_i^max - J_i^min) , r_i ∈ [0, 1]

A set of $N = 1000$ to $5000$ points are computed using Forward Kinematics (FK). Reach volume $V$ is approximated by measuring the coordinates limits:

Volume V ≈ (4/3)·π·R_max³ · (Valid Points / Total Sampled)
